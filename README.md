DisasterNetOS

A decentralized LoRa-based operating framework for resilient disaster communication.**

---

Overview

DisasterNetOS is an infrastructure-independent disaster communication framework designed to maintain reliable communication when conventional networks fail.

Built using ESP32 microcontrollers and SX1278 LoRa transceivers, the system creates a decentralized mesh network capable of transmitting emergency messages without relying on cellular towers, Wi-Fi, or the Internet.

Unlike traditional communication systems, DisasterNetOS introduces protocol-level features such as message prioritization, acknowledgements, duplicate suppression, store-and-forward forwarding, and fault tolerance to improve communication reliability during disasters.

---

Why DisasterNetOS?

Natural disasters frequently destroy communication infrastructure.

When this happens:

* Cellular towers stop working.
* Internet connectivity becomes unavailable.
* Survivors cannot send SOS messages.
* Rescue teams lose coordination.

DisasterNetOS was developed to solve this problem by creating a portable communication framework that continues operating even when existing infrastructure is unavailable.
